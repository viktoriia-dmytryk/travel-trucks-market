# 🚐 TravelTrucksMarket

**TravelTrucksMarket** is a responsive web application for browsing and renting
campers. Users can filter vehicles by various criteria, explore detailed camper
information, and find the perfect option for their trip.

## ✨ Features

- 🔍 **Filter campers** by:
  - Location
  - Vehicle type — _Alcove, Panel Van, Integrated, Semi Integrated_
  - Engine type — _Diesel, Petrol, Hybrid, Electric_
  - Transmission — _Automatic, Manual_
- ❤️ Add and remove campers from favorites
- 📄 Detailed camper page featuring:
  - Image gallery
  - Vehicle specifications
  - Customer reviews
  - Booking form
- ⭐ Ratings and review count display
- 📍 Camper location information
- 📱 Responsive design for mobile and desktop devices

## 🚀 Installation

**1. Clone the repository**

```bash
git clone https://github.com/viktoriia-dmytryk/travel-trucks-market.git
```

**2. Install dependencies**

```bash
npm install
```

**3. Start the development server**

```bash
npm run dev
```

The app will be available at `http://localhost:3000`.

---

## 📖 Usage

1. Browse the camper catalog.
2. Apply filters to find suitable vehicles.
3. Open a camper page to view detailed information.
4. Read customer reviews.
5. Submit a booking request using the booking form.
