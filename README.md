# 🚐 TravelTrucks

A catalog of rental campers and motorhomes with filtering by location, camper
form, engine type, and transmission.

## Features

- Filter by location, camper form (Alcove, Panel Van, Integrated, Semi
  Integrated), engine (Diesel, Petrol, Hybrid, Electric), and transmission
  (Automatic, Manual)
- Vehicle cards with photo, rating, review count, location, description, and key
  specs
- Pagination via "Load more" button
- Detailed card view via "Show more"

## Tech Stack

- **Next.js**
- **TypeScript**
- **CSS Modules**

## Getting Started

```bash
npm install
npm run dev
```
